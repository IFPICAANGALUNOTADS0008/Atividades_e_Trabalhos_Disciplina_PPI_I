Estruturando Páginas Web com HTML

O Que é HTML?

  • É uma das linguagens mais importantes dentro do Frontend.
  • Ela é usada para estruturar páginas web, definido o significado e a semântica de elementos e objetos dentro de um documento HTML.
  • Foi lançada em 1992, mas foi originalmente conceituada em 1989 pelo físico e cientista computacional, Tim Berners-Lee. Sua versão atual é o HTML5, lançada em 2014.

Estrutura Básica

A escrita básica do HTML é feita através de:
  • Elementos: Pares de palavras ou letras individuais, que marcam e limitam onde cada objeto da página web dever estar. É usado a tag de abertura, <palavra ou letra>, para marcar o inicio e a tag de fechameneto, </palavra ou letra>, para marcar o fim da localização de um objeto.
  • Atributos: Informações adicionais colocadas dentro de elementos, por exemplo, links para ir a outra página web, parâmetros de tamanhos de quadros ou definição de tipos de dados.
A estrutura básica de uma página web: 
             <!DOCTYPE html>
             <html>
               <head>
                 <title>Título da página</title>
               </head>
               <body>
                 <p>
                 Conteúdo da página
                 </p>
               </body>
             </html>
Função de cada acima tag:
  • !DOCTYPE html: Indentifica a versão do HTML usado para escrever o documento.
  • html e /html: Sinalizam o inicio e fim do documento HTML.
  • head e /head: Marcam o cabeçalho da página. Dentro dele é colocado o titulo da página, que aparece acima da barra de URL de alguns navegadores, metadados, entre outras coisas.
  • title e /title: Usado para marcar o titulo de uma página.
  • body e /body: Marcam o corpo da página. Entre eles ficam tudo aquilo que é visível no navegador.
  • p e /p: Demarcam o inicio e o fim de um parágrafo.

Elementos de Uso Diário

A seguir será mostrado os elementos normalmente usados e mais importantes dentro da linguagem HTML.

## Listas

Há três tipos de listas no HTML, sendo elas:

Listas Ordenadas
Nela cada objeto recebe uma numeração para indicar sua ordem. Ela é marcada pelos elementos <ol> e </ol> pra delimitar seu espaço, junto com <li> e </li>. p para indica onde cada item da lista deve ficar. Exemplo da estrutura:
                          <ol>
                            <li>Primeiro passo</li>
                            <li>Segundo passo</li>
                            <li>Terceiro passo</li>
                          </ol>
Listas Não Ordenadas
Usada quando não há uma ordem fixa de objetos, sendo sinalizadas por uma bola preta. É feita pelas tags <ul> e </ul>, para marcar seu espaço, e <li> com </li> para sinalizar a localização de cada item.Exemplo:
              <ul>
                <li>HTML</li>
                <li>CSS</li>
                <li>JavaScript</li>
              </ul>
Listas para definiçôes
Usada quando temos que estabelecer definições de coisas. Suas tags são <dl> e </dl>, para delimitar seu espaço, <dt> e </dt> para indicar um termo e <dd> e </dd> para fixar a descrição do termo.Exemplo:
              <dl>
                <dt>HTML</dt>
                <dd>Linguagem de marcação</dd>
                <dt>CSS</dt>
                <dd>Responsável pelo estilo</dd>
              </dl>

##Links

Links são usados para conectar páginas entre si. Eles permitem que o usuário navegue entre diferentes conteúdos na web.

Para criar um link, utilizamos a tag <a>, junto com o atributo href, que indica o destino do link.

Exemplo:

              <a href="https://www.exemplo.com">Acessar site</a>

O texto entre a abertura e fechamento da tag é o que será clicável.

##Imagens

As imagens são usadas para exibir conteúdo visual dentro da página.

Para inserir uma imagem, usamos a tag <img>. Essa tag não possui fechamento, sendo controlada apenas por atributos.

  • src: caminho da imagem
  • alt: descrição da imagem

Exemplo:

                <img src="imagem.jpg" alt="Descrição da imagem">

##Tabelas

Tabelas são usadas para organizar dados em linhas e colunas.

As principais tags são:

  • <table>: define a tabela
  • <tr>: define uma linha
  • <td>: define uma célula
  • <th>: define um cabeçalho

Exemplo:

                  <table>
                    <tr>
                      <th>Nome</th>
                      <th>Idade</th>
                      </tr>
                      <tr>
                      <td>João</td>
                      <td>20</td>
                    </tr>
                  </table>

##Formulários

Formulários são usados para coletar dados do usuário, como login, cadastro ou pesquisas.

A estrutura principal é feita com a tag <form>.

Dentro dela, podemos usar diferentes tipos de entrada:

  • Campos de texto
  • Senhas
  • Botões
  • Caixas de seleção

Exemplo:

               <form>
                 <input type="text" placeholder="Nome">
                 <input type="password" placeholder="Senha">
                 <input type="submit" value="Enviar">
               </form>
